# 📊 Hashtag Trend Analysis

## 📌 Project Overview
This project analyzes hashtag performance using Python.  
It calculates total engagement and visualizes the results using Matplotlib.

---

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib
- Git & GitHub

---

## 📂 Project Structure

```
Hashtag-trend-analysis/
│
├── data/
│   └── hashtags.csv
│
├── images/
│   └── hashtag_engagement.png
│
├── src/
│   └── analysis.py
│
└── README.md
```

---

## 📈 Output

The script generates:

- Total engagement per hashtag
- Bar chart visualization saved in `images/hashtag_engagement.png`

---

## ▶️ How to Run

1. Install dependencies:
```
pip install pandas matplotlib
```

2. Navigate to src folder:
```
cd src
```

3. Run the script:
```
python analysis.py
```
